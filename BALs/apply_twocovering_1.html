Some models will give just a single normalised spectrum (I/I0), in that case, you can just multiply it to the unabsorbed QSO spectrum to get synthetic FeLoBAL quasar spectrum¶

Set of two normalized spectrum:¶

  • I/I0 for emission lines
  • I/I0 for power-law continuum

Each component, power law emission and line emission, of unabsorbed QSO needs to be multiplied by corresponding normalized spectrum and then added back tohether to obtain the correct synthetic BAL quasar spectrum.¶

simpaqs provide various parameters used to generate unabsorbed qso spectrum with simqso. We will use those numbers to generate (1) power-law continuum model so that we can obtain (2) emission line model by subtracting the qso model by power-law model¶

Now apply the set of I/I0 to the quasar emission lines and power-law continuum separately¶

def ApplySimBALTwoCovering(z,wl_ii0,ii0s,wl,em,pl):
    ###Let's first redshift the absorption template
    ####First ii0 is for the line emission
    line_ii0=np.interp(wl,wl_ii0*(z+1),ii0s[0])
    ####Second ii0 is for the power law emission
    pl_ii0=np.interp(wl,wl_ii0*(z+1),ii0s[1])
    ###Now apply the two
    return em*line_ii0+pl*pl_ii0
